# Partner

Partner sind Vermittlungsagenturen, die eigene Freiberufler repräsentieren.

## Inhalt dieses Kapitels

- [Partner anlegen](anlegen.md)
- [Suchen und Filtern](suchen.md)

## Überblick

Jeder Partner-Datensatz enthält:

| Bereich | Inhalte |
|---------|---------|
| **Adresse** | Firma, Name 1, Name 2, Anschrift |
| **Konditionen** | Debitor-Nr, Kreditor-Nr, Kontaktsperre, Wiedervorlage |
| **Kommentar** | Freie Notizen |
| **Kontaktmöglichkeiten** | E-Mail, Telefon, Fax, Website, XING, GULP |
| **Zugeordnete Freiberufler** | Liste aller Freiberufler, die diesem Partner zugeordnet sind |
| **Kontakthistorie** | Chronologische Gesprächsnotizen |
