# Projekte

Projekte sind offene Stellen und laufende Aufträge, für die passende Freiberufler gesucht werden.

## Inhalt dieses Kapitels

- [Projekt anlegen](anlegen.md)
- [Positionen verwalten](positionen.md)
- [Freiberufler zuordnen](freiberufler-zuordnen.md)

## Überblick

| Bereich          | Inhalte                                                              |
|------------------|----------------------------------------------------------------------|
| **Allgemein**    | Projektnummer, Datum, Laufzeit, Status, Website-Sichtbarkeit         |
| **Beschreibung** | Kurzbeschreibung, ausführliche Beschreibung, Anforderungen           |
| **Einsatz**      | Einsatzort                                                           |
| **Zuordnung**    | Verknüpfter Kunde und/oder Partner (nach Anlage nicht mehr änderbar) |
| **Konditionen**  | Stundensatz VK, Debitor-Nr, Kreditor-Nr                              |
| **Freiberufler** | Zugeordnete Projektpositionen mit Status, Konditionen und Kommentar  |

## Projektstatus

| Status        | Bedeutung                    |
|---------------|------------------------------|
| **Offen**     | Aktiv besetzbar              |
| **Verloren**  | Auftrag ging an Mitbewerber  |
| **Canceled**  | Vom Kunden abgesagt          |
| **Besetzt**   | Alle Positionen sind besetzt |
| **Search zu** | Suche eingestellt            |
