Projekt 1: Microservices-Plattform für europäischen Versicherungskonzern
Zeitraum: 2023 – laufend (ca. 18 Monate)
Branche: Versicherungen
Rolle: Lead Backend Developer

Verantwortlich für den Entwurf und die Implementierung einer neuen Microservices-Plattform zur
Ablösung eines monolithischen Altsystems. Die Plattform verarbeitet täglich mehrere hunderttausend
Versicherungsanträge und -änderungen. Im Team von 8 Entwicklerinnen und Entwicklern übernahm die
Fachkraft die technische Führung sowie die Architekturentscheidungen. Einführung von Spring Boot 3.2,

mit Docker und Kubernetes. Test Coverage > 85% durch konsequentes TDD mit JUnit 5 und Mockito.

Eingesetzte Technologien: Java 21, Spring Boot 3.2, Spring Security, JPA/Hibernate, PostgreSQL,
Docker, Kubernetes, Maven, Git, GitHub Actions
