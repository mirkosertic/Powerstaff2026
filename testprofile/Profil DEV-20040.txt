Projekt 1: Management-Reporting-Plattform für einen Einzelhandelskonzern
Zeitraum: 2023 – laufend (ca. 13 Monate)
Branche: Handel / Retail
Rolle: Lead BI-Entwicklerin

Aufbau einer zentralen Management-Reporting-Plattform in Power BI auf Basis von Azure Synapse.
Entwicklung von 25+ Dashboards für Vertrieb, Einkauf, Logistik und Finanzen. DAX-Measures

SQL Server, Azure DevOps (Pipeline für Power BI-Deployment)

Rolle: Data Analyst / BI Developer

Microsoft Azure (grundlegend), Excel (für Legacy-Reports)
