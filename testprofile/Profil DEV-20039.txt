Projekt 1: Echtzeit-Zahlungsverarbeitungssystem für eine Direktbank
Zeitraum: 2022 – laufend (ca. 22 Monate)
Branche: Banking / FinTech
Rolle: Senior Java Developer / Softwarearchitekt

Mitverantwortlicher Architekt für ein neues Echtzeit-Zahlungsverarbeitungssystem (SEPA Instant
Credit Transfer). Event-Driven Architecture auf Basis von Kafka, Saga-Orchestrierung für

Eingesetzte Technologien: Java 21, Spring Boot 3, Spring Kafka, Apache Kafka,
MongoDB, PostgreSQL, Kubernetes, Prometheus, Grafana, Micrometer
